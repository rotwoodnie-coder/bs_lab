# 本地开发：影子资源 Registry ID（示例）

将本文件复制到仓库根目录，命名为 `README.local`（已加入 `.gitignore`，不会提交），并按本机数据库填写。

分步节奏与「何时填好再让 Agent 继续」见 **`docs/de-mock-handoff-checkpoints.md`**（Phase 1）。

## 三种规格（推荐）

| 用途 | 建议比例 | 本环境 `edu_media_registry.id` |
|------|----------|-------------------------------|
| 标准实验列表卡片（16:9 封面/视频区） | 16:9 | （填入数字 ID，建议名称：实验封面默认图） |
| 实验材料列表小图 | 1:1 | （填入数字 ID，建议名称：实验材料默认图） |
| 教材封面（竖版） | 竖屏 | （填入数字 ID，建议名称：教材默认图） |

## 联调约定

- 全组使用同一组 ID 时，由一人维护 MinIO 种子与登记行，并同步更新共享文档或内部 Wiki。
- 若 ID 失效，优先检查：MinIO 中对象是否存在、`storage_key` 是否与登记一致、租户与应用键是否与控制台一致。

## 校验

在控制台任意媒体预览或标准实验卡片中，使用上述 ID 应能正常出图或出流，无 404/403。
