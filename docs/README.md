## 维护责任人与更新规则

| 项目 | 说明 |
| --- | --- |
| 维护责任人 | 架构组 |
| 正文更新触发 | 路由、服务、仓库、前端页面、数据库迁移发生变化时同步更新正文 |
| 附录更新触发 | 新增术语、新增错误码、新增图示或需要对外解释的内容出现时更新附录 |
| 归档规则 | 旧版本正文进入 `archive/`，当前有效版本保留在 `core/` |
| 版本命名 | 当前版本使用 `v1.1`，归档版本使用 `v1.0` 等历史版本号 |
| 审阅规则 | 任何接口或数据库结构变更，必须同步检查正文、附录与变更记录 |
