# 错误码参考

| 错误码 | 含义 | 说明 |
| --- | --- | --- |
| `CONTENT_TOO_LONG` | 内容过长 | 实验步骤或富文本内容超长 |
| `EXP_NAME_EMPTY` | 实验名称不能为空 | 创建或保存实验时校验失败 |
| `NOT_FOUND` | 资源不存在 | 目标记录不存在 |
| `INTERNAL_ERROR` | 内部错误 | 未捕获异常 |
| `400` | 参数校验失败 | 请求参数不合法 |
| `401` | 未授权 | 缺少用户身份 |
| `404` | 资源不存在 | 目标接口或记录不存在 |
| `4001` | 实验步骤内容过长 | `CONTENT_TOO_LONG` 的路由映射 |
| `4002` | 实验名称不能为空 | `EXP_NAME_EMPTY` 的路由映射 |
| `5000` | 服务内部错误 | 未知异常 |
